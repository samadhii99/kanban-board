🚀 Kanban Board - Task Management Made Easy!
A powerful and interactive Kanban board built with React, TypeScript, and Vite. Designed for agile project management, this board helps teams stay organized and track progress with ease.

✨ Features:
-Drag and Drop: Seamlessly move cards between columns using the @dnd-kit library.

-Task Details: Add a title, description, labels (Bug, Feature, Enhancement, Documentation, Priority Levels, Frontend), due date, and mark tasks as complete.

-Collaborative Workspace: Engage team members by adding them to cards using their email addresses.

-Customizable Labels: Easily categorize tasks with predefined labels to maintain clarity.

-Responsive Design: Sleek, user-friendly interface that works on any device.

-Built with Modern Tools:

  -React + TypeScript for robust frontend development.

  -Vite for fast builds and smooth development experience.

  -Redux for state management.

  -Ant Design for a polished UI.

💡 Use Cases:
-Software Development: Track bugs, features, and enhancements efficiently.

-Team Collaboration: Assign tasks and track progress with team member involvement.

-Project Management: Keep tasks organized and easily movable across phases.
